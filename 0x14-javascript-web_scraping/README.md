JavaScript Web Scraping
Overview
Explore JavaScript's power in web scraping led by CTO Guillaume from Holberton School. This project dives into JSON data manipulation, request and fetch APIs, and file handling with the fs module.

Objectives
Master JavaScript's versatility.
Efficiently handle JSON data.
Use request and fetch APIs for web scraping.
Read and write files using the fs module.

Guidelines
Editors: vi, vim, emacs
Platform: Ubuntu 20.04 LTS, node (v14.x)
Formatting: End files with a new line; start with #!/usr/bin/node
README.md: Mandatory at the project root
Code Style: Semistandard compliant (Standard + semicolons on top)
Executables: All files must be executable
File Length: Code length tested using wc
var Usage: Not allowed

Installation
Node 14:
- curl -sL https://deb.nodesource.com/setup_14.x | sudo -E bash -
- sudo apt-get install -y nodejs
Semi-standard:
- sudo npm install semistandard --global

Request Module:
- sudo npm install request --global
- export NODE_PATH=/usr/lib/node_modules
